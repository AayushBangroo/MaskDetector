COVID-19 Mask Detector

The aim of the project is to develop an application capable of detecting masks on a person's face. This application operates in two modes:

  • Live Capture
  • Image Capture
In the Live Capture mode mask detection is done on a video stream which is captured using the camera and in the second mode Image Capture mask detection is done on an image uploaded by the user. Click on the below button to head over to mode selection.
